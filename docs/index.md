# 功能介绍

## 概要

WatchAlert 是基于Go+React开发的监控告警管理平台。可以完全替代 `AlertManager`、`PrometheusAlert(通知平台)` 等组件，支持配置交互式通知、通知对象、值班系统和聚合功能，并且拥有规则管理、告警抑制、告警推送和告警静默能力。可以提升运维效率，降低维护成本。
